How to use CSS

First, you can use <style> tag in <head> section. But I prefer to use <link> tag instead and write your code on another file. It is more easy to read and edit.

<head>
    <link rel="stylesheet" href="your css file path">
</head>

You can use CSS by following these 3 steps.
1. select tag / id / class
2. open block '{  }'
3. write 'attribute: value1 value2 value3....;'

Because there are so many attributes and values, I'm gonna explain how to use selector in the first step.

Selector - class

We can confront so many cases during web programming. So there are also many selectors. But to make it simple, I'll show you how to use the most important and widely used way, 'class'.

In order to use 'class', in html, you should write like the example below.

<div class="container">

You should use any other tags instead of <div> and also any other class names instead of container.
Next step is to select that class name in css file like this. Only thing you have to do is writing '.' in front of the class name and then write what features you would change.

.container {
    attribute: value1 value2 value3...;
}
